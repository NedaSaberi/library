# library
This is a library project including the following features:
- Adding a new member
- Adding a new book
- Editting a member
- Editting a book
- Removing a member
- Removing a book
- Searching a member by name
- Searching a member by code
- Searching a book by name
- Searching a book by code
- Lending a book to a member
- Reporting by book
- Reporting by member
